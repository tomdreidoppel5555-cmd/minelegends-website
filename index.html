Voting & Votifier

How voting & Votifier work ‚Äî and where to put IP/Port/PublicKey.

Votifier Settings (example)

Votifier IP / Host: minelegends.mc-game.xyz

Votifier Port: 25020

You must paste the server's public key (the Votifier public key) into your voting panel so votes can be delivered. In most server list panels you will find fields labeled "Votifier IP/Host", "Votifier Port" and "Public Key". Make sure the listener on your server is reachable from the vote provider (no firewall blocking, correct port forwarded). After that the vote site will send votes to your server automatically.

Where to find the Votifier info

  1. On your server control panel (where you installed Votifier), there is a generated public/private key pair. Copy the Public Key (text block) and paste it into the server list "Public Key" field.
  2. Set Votifier IP/Host to your server's public hostname (example: minelegends.mc-game.xyz) or to the IP where the votifier listener is bound.
  3. Set Votifier Port to the port configured in the Votifier plugin (example: 25020). Ensure the port is open in firewall/host settings.

Once saved, your voting site should display "Votifier settings saved" ‚Äî then vote deliveries will work and the plugin will grant the reward when a vote arrives.

API / ServerKey for votes (example)

If your server list provides an API key (ServerKey) you can query votes with the API endpoints. Example endpoints:

GET server details:
https://minecraft-server.eu/api/v1/?object=servers&element=detail&key={ServerKey}

GET votes list:
https://minecraft-server.eu/api/v1/?object=servers&element=votes&key={ServerKey}

Replace {ServerKey} with your API key from the server-list panel. Usually the key is shown on your server entry page under "API Serverkey".
